Instructions to use indexer.py
1. Open terminal
2. Type python3 indexer.py
3. Terminal will show current file/json that is getting parsed from DEV folder
4. After program is done parsing through all jsons, inverted_index.csv and token_row_map.json files will be created in main directory



Instruction to use searcher.py:
1. Open terminal
2. Type python3 searcher.py 
3. The terminal will output a local host link you can type into your browser
http://localhost:5000  (Can launch automatically in some cases)
4. You can paste this into your browser and start querying the system
5. The results will display themselves on the results page, to use search again, click back page button
